  3. Inspect element: Inspecting a webpage's code can be fun, and in fact, there is a way to do it. You can right click on the element you want to see the code of, and press Inspect Element or Inspect. This will bring up lots of code. Congratulations! Now you can browse a website's code. Now, if you want to get to that page even faster, you can use a short cut. Press Command+Option+i on a Mac or Ctrl+Shift+i on Windows. This will bring up a JavaScript console, where you can run JavaScript. Go to the top and click Elements.
